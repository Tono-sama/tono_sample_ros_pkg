# tono_simple_communication
topic通信をしよう！

## simple_talker_node
- topic通信の送信側のノード
- 実行コマンド:  
`rosrun tono_simple_communication simple_talker_node`
- 端末にhelloworldが表示されることを確認できればok!

## simple_listener_node
- topic通信の受信側のノード
- 実行コマンド:  
`rosrun tono_simple_communication simple_listener_node`
- 端末にhelloworldが表示されることを確認できればok!
